# Plan 157 — apps/common cleanup: UserSchema extraction + dead glob removal

## Context

Plan 155 restructured the monorepo into `framework/` and `apps/`, creating `@arbor/app-common` for
shared app schemas. Two small items were left incomplete:

1. `apps/api/src/routes/users.router.ts` could not move to `apps/common` because it imports
   `UserSchema` from `../repositories/users.repository.js` — a local api file.
2. `pnpm-workspace.yaml` still has a `packages/*` glob; the `packages/` directory no longer exists.

## Goal

- Extract `UserSchema` to `apps/common` so `users.router.ts` can follow.
- Remove the dead `packages/*` glob from `pnpm-workspace.yaml`.
