# Plan 177 — MUI Foundation

## Context

The ledger UI at `apps/ui/src/ledger/LedgerTable.tsx` is a bare HTML `<table>` with
inline styles. Plans 178–180 replace it with a polished MUI-based UI. This plan installs the
required dependencies and creates the shared dark theme so every subsequent plan can import from
a single source of truth.

No store or reducer changes. No new components beyond the theme file and provider wiring.

## Goal

1. Add `@mui/material`, `@mui/icons-material`, `@emotion/react`, `@emotion/styled`,
   `react-markdown`, `remark-gfm`, and `rehype-highlight` to `apps/ui/package.json`.
2. Create `apps/ui/src/theme.ts` — exports a single MUI `Theme` built with `createTheme`
   using `mode: 'dark'`, a near-black background (`#0d0d0d`), and a slate primary
   (`#4a9eff`). Keep the palette minimal — no custom component overrides yet.
3. Wrap `apps/ui/src/main.tsx` (or the root JSX element in `App.tsx`) in
   `<ThemeProvider theme={theme}>` + `<CssBaseline />`. `CssBaseline` resets browser
   defaults consistently.

Nothing else changes. Plans 178+ consume the theme.
