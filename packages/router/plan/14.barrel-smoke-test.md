# PLAN.14 — Add barrel export smoke test

## Problem

`index.ts` defines the public API surface. There is no test that verifies
the exports are importable and correct. If an export is accidentally removed,
only downstream consumers would catch it.

## Change

1. Create `src/index.test.ts`.
2. Import every public symbol from `./index.js`.
3. Assert that each function export is a function and each type export
   resolves (use `expectTypeOf` for types, runtime checks for values).

Example:

```typescript
import { defineRoutes, route, section, httpRoute, createServer, createClient, openApiRoute, generateSpec } from './index.js';
import type { RouteNode, InferRoute, InferContext, ChildUnion, CtxMap, Derive, Flatten, HttpContext, HttpMethod, HandlerMap, FetchLike, OpenApiContext, OpenApiMeta } from './index.js';

it('exports all public functions', () => {
  expect(typeof defineRoutes).toBe('function');
  expect(typeof route).toBe('function');
  // ...
});
```

## Risk

Near zero. Additive test.
