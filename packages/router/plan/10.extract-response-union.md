# PLAN.10 — Extract `ResponseUnion` to a shared location

## Problem

The `ResponseUnion` type is defined identically in both `server.ts` and
`client.ts`:

```typescript
type ResponseUnion<Resp> = {
  [S in keyof Resp]: { status: S; body: Resp[S] };
}[keyof Resp];
```

## Change

1. Move `ResponseUnion` to `route-node.ts` (or a new `types.ts` if it doesn't
   fit there thematically) and export it.
2. Import it in `server.ts` and `client.ts`.
3. Optionally export it from the barrel if consumers need it.

## Risk

Near zero. Mechanical dedup.
